## Instructions

### At start
- ⏎ Enter: select / exit  
- ⬆️ Up arrow: up  
- ⬇️ Down arrow: down  

### In-game
- ⬅️ Left arrow: move left  
- ➡️ Right arrow: move right  
- ⬆️ Up arrow: climb ladders  
- ␣ Space: jump  

### Objective
🏆 Reach the highest platform to rescue Princess Peach 👑
